Materia:Desarrollo de Aplicaciones Web

En esta unidad veremos los fundamentos del desarrollo web

Tendremos dos temas:

Tema 1: Introducción al Desarrollo Web

Tema 2: Estructura y diseño de páginas web

Subtema 1.1.2: Herramientas y tecnologías esenciales:
editores, navegadores y servidores locales.
La tarea consistirá en: Familiarizarse con las herramientas
esenciales para el desarrollo web mediante la instalación y uso
básico de un editor de texto, un navegador web y un servidor local
para crear y probar una página web estática.
